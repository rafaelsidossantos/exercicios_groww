O que é html?

HTML (Linguagem de Marcação de HiperTexto) é a ferramenta mais básica para estruturar e dar significado a um conteúdo da web.

Antes da existência do HTML, pessoas necessitavam digitar todo o endereço para serem direcionadas a uma página da Web. No entanto, com o advento do HTML, direcionar páginas a outras ficou muito mais fácil e dinâmico através do "Hipertexto". Além disto, tal ferrameta, conta com outras ferramentas, tais como, o CSS que é utlizada para o layout da página web.

O HTML funciona através de tags, as quais são representadas pelos sinais "<" e ">" e um elemento entre eles sinais, o qual indicará a forma que o texto deve se comportar.
